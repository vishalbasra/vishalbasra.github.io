# Things to know

**Common Chef things to know**

| ------------- |:-------------:|
| `Omnibus`      | The open-sourced tool that Chef uses to create installers with scripting engine payloads (their own bundle of Ruby) |
| `/opt/chefdk`      | Almost all of the ChefDK stuff! |
